# Block Library Template

将论文沉淀成组块库时，优先用下面的条目格式：

```markdown
## 组块名称

- 来源论文：
- 组件 ID：
- 问题类型：
- 核心技术手段：
- 适用前提：
- 典型失败模式：
- 可替换实现：
- 推荐复用场景：
- 不适用场景：
- 关联组块：
- 证据锚点：
```

目录建议：

```text
papers/
  topic-name/
    paper-name.md
blocks/
  objective/
  architecture/
  optimization/
  training/
  data-processing/
  evaluation/
```
